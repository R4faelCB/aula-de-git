# Projeto Java usando Git e GitHub

## Comandos basicos do Git
```bash
echo "# aula-de-git" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:R4faelCB/aula-de-git.git
git push -u origin main
```
